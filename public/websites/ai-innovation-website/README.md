# AI Innovation

Diese Website wurde mit dem AI Website Generator erstellt.

## Netlify Deployment

1. Gehe zu [netlify.com](https://netlify.com)
2. Klicke auf "Add new site" → "Import from Git"
3. Wähle dieses Repository
4. Build Settings:
   - **Publish directory**: `public/websites/ai-innovation-website`
5. Klicke "Deploy site"

## Automatische Updates

Änderungen im AI Website Generator werden automatisch zu diesem Repository gepusht.
Netlify deployed automatisch bei jedem Push.
